## 3. Ctrl+Z的使用  
&emsp;&emsp;Ctrl+Z主要是将线程挂起，但并不结束进程（和Ctrl+C有区别）。Ctrl+Z是挂起，然后可以干别的事情，等别的事情干完了，再输入命令fg即可回到被挂起的进程上面。
```powershell
fg
```
